## Capturing them

- Browser window around **1440px** wide; hide the bookmarks bar.
- Wait for scroll-reveal animations to finish before capturing.
- On Windows: `Win + Shift + S`, then paste into Paint and save as PNG.
- For a full-page shot: DevTools → `Ctrl+Shift+P` → "Capture full size screenshot".
- Mobile views: DevTools device toolbar (`Ctrl+Shift+M`) → iPhone 14 Pro.

Keep each file under ~500 KB so the repo stays light.
